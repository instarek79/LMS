Data is information that is collected, stored, and analyzed. It can be anything from numbers and text to images, sounds, and videos. Data comes in various forms and can be structured, semi-structured, or unstructured.

Key Points:

  • Structured Data: Organized and easily searchable, like data in tables (e.g., Excel spreadsheets).
  • Semi-Structured Data: Has some organizational properties but not as rigid as structured data (e.g., XML, JSON).
  • Unstructured Data: Lacks a specific format or structure (e.g., text documents, social media posts).

10.2 Types of Data

Now, let's explore the different types of data. Data can be broadly categorized into quantitative and categorical data.

 10.2.1: Quantitative Data

Quantitative data represents quantities and is numerical. It's data that can be measured and counted. Quantitative data can be further divided into continuous and discrete data.

Continuous Data: Continuous data can take any value within a range. For example, the height of a person, the temperature outside, or the time it takes to run a marathon. These values can be decimals and can be infinitely precise.

Discrete Data: Discrete data, on the other hand, consists of distinct and separate values. For example, the number of students in a class, the number of cars in a parking lot, or the number of books on a shelf. These values are whole numbers and can't be subdivided meaningfully.

 Can you identify whether the following data is continuous or discrete?

The number of siblings you have.
Your body weight.
The score in a basketball game.
 

Perfect! The number of siblings and the score in a basketball game are examples of discrete data, while body weight is continuous data.

10.1.2 Categorical Data

 Categorical data represents characteristics and can be divided into categories. This type of data is often descriptive and can be split into nominal and ordinal data.

Nominal Data: Nominal data is used for labeling variables without any quantitative value. For example, the type of car you drive, your favorite color, or the city you live in. There's no inherent order in nominal data.

Ordinal Data: Ordinal data, however, does have a meaningful order, but the differences between the values are not necessarily consistent. For example, customer satisfaction ratings (satisfied, neutral, dissatisfied), educational level (high school, bachelor's, master's), or ranks in a competition (first, second, third).

 Can you categorize the following as nominal or ordinal?
