5.1 Introduction to Matplotlib
Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python. It's widely used for data visualization, especially in data science and machine learning. The library is incredibly versatile, allowing you to create a variety of plots such as line plots, bar plots, histograms, and more. Think of it as your go-to tool for turning raw data into beautiful graphs.

Using Matplotlib

Most of the Matplotlib utilities lies under the pyplot submodule, and are usually imported under the alias plt.

import matplotlib.pyplot as plt

Creating a Simple Line Plot

Let's start with a simple line plot. Suppose we have some data representing the sales of a product over six months. Here's how we can visualize it:

import matplotlib.pyplot as plt



# Data

months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']

sales = [100, 120, 90, 150, 170, 130]



# Creating the plot

plt.plot(months, sales)



# Adding titles and labels

plt.title('Monthly Sales')

plt.xlabel('Month')

plt.ylabel('Sales')



# Displaying the plot

plt.show()

Can you think of another scenario where a line plot would be useful? Maybe to track your monthly expenses or your weight over time? 

Now, let's add some customization to our plot. We can change the color, add markers, and even customize the line style. Here's an example:

plt.plot(months, sales, color='green', marker='o', linestyle='dashed')



plt.title('Monthly Sales')

plt.xlabel('Month')

plt.ylabel('Sales')



plt.show()

 

 

What's your favorite color for plots? Do you prefer something bright like red or something more subdued like blue? 

Creating Bar Plots

Bar plots are great for comparing categories. Let's create a bar plot to show the sales of different products:

products = ['Product A', 'Product B', 'Product C']

sales = [300, 150, 200]



plt.bar(products, sales, color=['red', 'blue', 'green'])



plt.title('Sales by Product')

plt.xlabel('Product')

plt.ylabel('Sales')



plt.show()

 

 

What other types of data do you think would look good in a bar plot? Maybe comparing different countries' populations or different types of fruits' prices?

Creating Histograms

Histograms are useful for showing the distribution of a dataset. Let's say we have some data on the ages of a group of people. We can visualize it like this

ages = [23, 25, 31, 35, 45, 33, 25, 36, 29, 28, 22, 21, 34, 42, 43, 40]



plt.hist(ages, bins=5, color='purple')



plt.title('Age Distribution')

plt.xlabel('Age')

plt.ylabel('Frequency')



plt.show()

 

 

 

Think about the last time you saw a histogram. What was it showing? Maybe it was the distribution of scores in a class or the frequency of different word lengths in a text.

Creating Scatter Plots

Scatter plots are perfect for showing relationships between two variables. Here's an example using some random data:

import numpy as np



x = np.random.rand(50)

y = np.random.rand(50)



plt.scatter(x, y, color='orange')



plt.title('Random Scatter Plot')

plt.xlabel('X-axis')

plt.ylabel('Y-axis')



plt.show()

Can you think of any datasets where a scatter plot would be useful? Maybe to show the relationship between height and weight, or study time and grades?

5.2 Matplotlib with Pandas

Line Plot with Pandas

Let's start with a simple line plot using pandas. Suppose we have a dataset representing the sales of two products over six months. We'll first create a DataFrame and then plot it:

import pandas as pd
import matplotlib.pyplot as plt

# Creating a DataFrame
data = {
    'Month': ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'],
    'Product_A': [100, 120, 90, 150, 170, 130],
    'Product_B': [80, 110, 100, 130, 140, 160]
}
df = pd.DataFrame(data)

# Plotting the data
df.plot(x='Month', y=['Product_A', 'Product_B'], kind='line', marker='o')

plt.title('Monthly Sales for Products A and B')
plt.xlabel('Month')
plt.ylabel('Sales')

plt.show()

 

Can you think of other scenarios where you might use a line plot with pandas? Maybe to track temperature changes over time or stock prices?

Bar Plot with Pandas

Next, let's create a bar plot using the same data. Bar plots are great for comparing different categories:

df.plot(x='Month', y=['Product_A', 'Product_B'], kind='bar', color=['blue', 'orange'])



plt.title('Monthly Sales for Products A and B')

plt.xlabel('Month')

plt.ylabel('Sales')



plt.show()

 

 

What kind of data do you usually visualize with bar plots? Perhaps sales data, survey results, or even election results?

Histogram with Pandas

Histograms are useful for showing the distribution of a dataset. Let's create a histogram of ages using pandas

 

ages = [23, 25, 31, 35, 45, 33, 25, 36, 29, 28, 22, 21, 34, 42, 43, 40]

df_ages = pd.DataFrame({'Age': ages})



df_ages.plot(kind='hist', bins=5, color='purple')



plt.title('Age Distribution')

plt.xlabel('Age')

plt.ylabel('Frequency')



plt.show()
