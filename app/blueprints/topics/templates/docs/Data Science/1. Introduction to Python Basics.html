1.1 Python Variables

First, let's talk about variables. In Python, a variable is used to store information. You can think of it as a container that holds data. 

For example, you might want to store a number or a piece of text in a variable.

Example:

# Example of a variable
 name = "Alice" 
 age = 25

In this example, name is a variable holding the string 'Alice', and age is a variable holding the number 25. Variables can hold different types of data, which leads us to our next topic: Data types

1.2 Python Data Types

Python supports various data types, including integers, floats, strings, and booleans. Each type of data serves a different purpose

Example:

# Examples of data types
integer_number = 10        # Integer
floating_number = 10.5     # Float
text = "Hello, World!"     # String
is_active = True           # Boolean

As you can see, integer_number holds an integer, floating_number holds a float, text holds a string, and is_active holds a boolean value.

Now, let's move on to Functions.

 

1.3 Python Functions

Functions are blocks of code that perform a specific task. They help us to reuse code and make our programs more organized. You can define a function using the def keyword.

Example:

# Example of a function
def greet(name):
    return f"Hello, {name}!"

# Calling the function
print(greet("Alice"))

In this example, we defined a function called greet that takes a parameter name and returns a greeting message. When we call greet("Alice"), it prints 'Hello, Alice!

Now, let's talk about  Conditions.

1.4 Python Conditions

Conditions allow us to execute certain code based on whether a condition is true or false. We use if, elif, and else statements to create conditions."

Example:

# Example of a condition
age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Here, we check if age is greater than or equal to 18. If it is, we print 'You are an adult.' Otherwise, we print 'You are a minor.' Now, let's find about Loops

1.5 Python Loops

Loops allow us to repeat a block of code multiple times. Python has two main types of loops: for loops and while loops.

Example:

# Example of a for loop

for i in range(5):

    print(i)



# Example of a while loop

count = 0

while count < 5:

    print(count)

    count += 1

In the first example, the for loop iterates over a range of numbers from 0 to 4 and prints each number. In the second example, the while loop continues to print count until count is less than 5, incrementing count by 1 each time.
